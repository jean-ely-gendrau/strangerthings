Bienvenue dans le Fan Lab de Stranger Things

Ce mini site te permet de comprendre comment fonctionne la réécriture d’URL avec Plesk / Apache (mod_rewrite).

  • Les fichiers réels sont rangés dans /strangerthings/
  • Les URLs visibles sont propres en /hawkins/…
  • Le tout est piloté par des règles dans .htaccess
Voir les personnages Explorer les lieux

Exemple de réécriture d’URL

Regarde comment une règle de réécriture transforme l’URL :

# URL vue par l’utilisateur
/hawkins/personnages/      →   /strangerthings/personnages.html

# Dans .htaccess
RewriteRule ^hawkins/personnages/?$ /strangerthings/personnages.html [L]
